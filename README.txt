Social Dating Matching and Clustering.

This is a repository of a project on social dating using machine learning techniques.

There are sub directories that support data ETL pipeline, as well as the code itself.

Sub Repositories
————————————————
(1) Code: consist a package of machine learning algorithms including KNN, SVM (Pegasus), Perceptron (and its variations) , K-means clustering, Naive Bayes Clustering, and Boosting (AdaBoost)

(2) Data: includes the raw data and source codes for sampling data for cross validation

(3) src: contains a CSVReader.java for parsing data into usable (quantified) data.

(4) Write_up: contains the full project documentation from abstract to conclusions.
